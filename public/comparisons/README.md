## Key Refinements Applied

All French translations include the following refinements from the localization review:

1. **"Cash" branding** - Using "Wealthsimple Cash" consistently instead of "Comptant"
2. **"valide en ce moment"** - Instead of "fonctionnel actuel" for current referral code
3. **"généralement"** - Instead of "typiquement" throughout
4. **"depuis une grande banque"** - Correct preposition for transfer context
5. **"Compte Cash (chèques)"** - Hybrid term for clarity
6. **OCRI** - Updated from OCRCVM for the regulatory body name
7. **Typography** - Proper French spacing (espace insécable before : ; ? !)
8. **"fiable et sécuritaire"** - For "safe and reliable"
