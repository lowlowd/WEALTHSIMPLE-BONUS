## But Why Does Wealthsimple Have This 7-Day Rule?

## Frequently Asked Questions

### I opened my account months ago but never funded it. Can I still add a code?

Yes. Wealthsimple distinguishes between "account opening" and "activation." If you opened an account 6 months ago but never deposited a single dollar, you are still considered a new client. You can enter a referral code at any time before you fund. Once you apply a code, you have 30 days to fund‚Äîbut if that window lapses, the code simply expires and you can reapply the same (or a different) code for another 30-day window. There's no limit while your account remains unfunded.

### Can customer support add the code for me if I missed the 7-day window?

Generally, no. Wealthsimple‚Äôs system is designed to hard-lock the referral option exactly 168 hours (7 days) after your first deposit lands. Customer support agents typically do not have the ability to override this hard-coded limit, so it is critical to act immediately.

### I clicked a referral link to sign up. Do I still need to enter the code manually?

Usually, no‚Äîthe link should apply the code automatically. However, to be safe, you should always verify. Open the app, go to the Referrals section (gift icon), and look for a banner that says "You were referred by [Name]." If you don't see it, enter the code 9C6DMQ manually just to be sure.

### Why am I getting an "invalid code" error?

This usually happens for one of two reasons:

1. **You are not a new client:** If you had an old Wealthsimple account (even a closed one) linked to your email or SIN, you are not eligible for a new client bonus.
2. **The 7-day window has passed:** If you funded your account more than 7 days ago, the system will reject any new codes.

### I added the code and funded $1. Why haven't I received my bonus yet?

While Wealthsimple aims to pay the base $25 bonus within 24 hours of funding, it can sometimes take up to 5-10 business days if the system needs to verify your deposit.[^3] Larger bonuses (like the $1000 Generation bonus) often take longer to process, typically landing within 14 days after your funding period ends.

### What if I made a mistake entering my referral code?

If you entered an incorrect code that was accepted, you'll need to contact Wealthsimple support to see if they can correct it. This is easier to fix than adding a new code after the window closes.

### Will I lose my sign up bonus if I withdraw money?

If you withdraw funds within 180 days of receiving your Wealthsimple sign up bonus, the bonus amount may be deducted from your withdrawal.[^1]
